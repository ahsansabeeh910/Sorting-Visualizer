## Features

Multiple Sorting Algorithms
Real-Time Visualization
Pause / Resume / Stop Controls
Statistics Tracking
Complexity Display
Manual Array Input
CSV Export
Theme Switching
Progress Tracking
Threaded Execution

---

## Supported Algorithms

| Algorithm      | Complexity |
| -------------- | ---------- |
| Bubble Sort    | O(n²)      |
| Selection Sort | O(n²)      |
| Insertion Sort | O(n²)      |
| Merge Sort     | O(n log n) |
| Quick Sort     | O(n log n) |
| Heap Sort      | O(n log n) |
| Shell Sort     | ~O(n^1.5)  |

## Installation

```bash
git clone https://github.com/ahsansabeeh910/Sorting-Visualizer
pip install -r requirements.txt
```

---

## Run Project

```bash
python sorting_visualizer.py
```

---

## Controls

* Generate → Create New Array
* Load → Manual Input
* Start → Begin Sorting
* Pause / Resume → Control Animation
* Stop → Terminate Sorting
* Export → Save Stats CSV
